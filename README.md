# Shadow of the Forgotten Forest

Визуальная новелла, созданная на движке Ren'Py.

## Как запустить игру

### Способ 1: Через Ren'Py (для разработки)

1. **Скачай и установи Ren'Py**
   - Перейди на официальный сайт: https://www.renpy.org/
   - Скачай последнюю версию (рекомендуется 8.x.x)
   - Распакуй архив в удобное место

2. **Склонируй репозиторий**
   ```bash
   git clone https://github.com/ТВОЙ_НИК/Shadow-of-the-Forgotten-Forest.git
   ```

3. **Перемести папку с игрой**
   - Открой папку, куда ты распаковал Ren'Py (например, `renpy-8.5.2-sdk`)
   - Скопируй туда папку `Shadow of the Forgotten Forest` (которую ты только что склонировал)
   - **Важно:** папка с игрой должна находиться **на одном уровне** с `renpy.exe` и папками `game`, `launcher` и т.д.
